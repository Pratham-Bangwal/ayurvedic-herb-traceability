# Setup Guide

## Prerequisites
- Node.js LTS
- npm 9+
- (Planned) MongoDB running locally
- (Planned) Hardhat for smart contracts

## Install Dependencies
At repo root:
```
npm install
```

## Run Backend (placeholder without DB)
```
npm run dev:backend
```
